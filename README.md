# ShopManage

## UI & Design Standards
To maintain design uniformity across the application, all new screens **must** follow the standards defined in:
- **AppBar Title**: Use `CustomAppBarTitle` in `lib/ui/common/app_bar_title.dart`.
- **Global Actions**: Use `AppBarActions` in `lib/ui/common/app_bar_actions.dart`.
- **Layout**: titles must be centered (`centerTitle: true`) and use `leadingWidth: 96` for the Back + Menu row.

Refer to `.agent/workflows/new_screen_ui_standards.md` for a full skeleton and implementation guide.
